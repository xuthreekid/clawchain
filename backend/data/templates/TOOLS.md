# TOOLS.md - 本地备注

Skills 定义工具用法。此文件存放**你的**环境特定信息 — 仅属于你本地的配置与约定。

## 适合放什么

例如：

- 摄像头名称与位置
- SSH 主机与别名
- TTS 偏好语音
- 扬声器/房间名称
- 设备昵称
- 任何环境相关的内容

## 示例

```markdown
### 摄像头

- living-room → 主区域，180° 广角
- front-door → 入口，运动触发

### SSH

- home-server → 192.168.1.100, user: admin

### TTS

- 偏好语音: "Nova"（温暖、略带英式）
- 默认扬声器: Kitchen HomePod
```

## 为何分开？

Skills 是共享的。你的配置是你自己的。分开存放意味着更新 Skills 不会丢失你的笔记，分享 Skills 也不会泄露你的基础设施。
